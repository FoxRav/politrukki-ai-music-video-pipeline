# 08 — Editointi & koonti

**Työkalu:** DaVinci Resolve

## Tehtävät
- [ ] Projekti luotu, asetukset (resoluutio, fps) säädetty
- [ ] Audio synkattu timelinelle (stems vaiheesta 07)
- [ ] Klipit leikattu rytmiin (lähde: `06_decode/davinci_ready/`)
- [ ] Siirtymät ja efektit lisätty
- [ ] Color grading tehty
- [ ] Lopputekstit / brändäys lisätty
- [ ] Final render exportattu (MP4 1080p / 4K)

## Alikansiot
| Kansio      | Sisältö                                              |
|-------------|------------------------------------------------------|
| `project/`  | DaVinci Resolve `.drp` ja muut projektitiedostot     |
| `exports/`  | Välirenderit, preview-tarkistukset                   |
| `final/`    | Lopullinen master-render → menee vaiheeseen 09       |

## Render-asetukset (suositus)
- Codec: H.264 / H.265
- Resoluutio: 1920x1080 tai 3840x2160
- FPS: 24 / 25 / 30 (vakio läpi pipelinen)
- Audio: AAC 320 kbps tai PCM
