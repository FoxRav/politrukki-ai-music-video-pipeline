# 03 — Audio & Laulu

**Työkalu:** ElevenLabs

## Tehtävät
- [ ] Äänimalli / voice valittu
- [ ] Instrumentaali / musiikkipohja generoitu
- [ ] Laulu generoitu lyriikoista
- [ ] Audio-mixaus ja master valmis
- [ ] Lopullinen audiotiedosto exportattu (WAV/MP3)

## Alikansiot
| Kansio            | Sisältö                                          |
|-------------------|--------------------------------------------------|
| `voice/`          | Voice-mallit, sample-pätkät, ID-muistiinpanot    |
| `instrumentaali/` | Generoidut instrumentaalipohjat (raw)            |
| `vocals/`         | Generoidut vokaaliraidat (raw)                   |
| `mix/`            | Mix-työtiedostot (DAW-projektit, bouncet)        |
| `master/`         | Lopullinen master-tiedosto → menee vaiheeseen 07 |

## Huom.
Audio-tiedostot ovat oletuksena `.gitignore`d — säilytä paikallisesti.
